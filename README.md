# 📚 Dizionario Parole Trentine
 
Questo progetto contiene i file di un sito web che permette di cercare parole in dialetto trentino dentro un database excel, in modo semplice e veloce.

Analisi dei requisiti (RE) disponibile [qui](https://docs.google.com/document/d/1UwrbHYbU9CWfF6wtS21rGHwb0z-wfi7FUpjAFH0owpM/edit?usp=sharing).

Anteprima Wireframe del sito disponibile [qui](https://www.canva.com/design/DAG-kqtxfuw/HpPp_k9ZHGJXRMVXBaVcbA/edit?utm_content=DAG-kqtxfuw&utm_campaign=designshare&utm_medium=link2&utm_source=sharebutton)

## 🚀 Obiettivo del progetto

L’idea è creare un sito pubblico dove chiunque possa:

*   Cercare parole in dialetto trentino
*   Vedere traduzione e significato
*   Avere uno strumento per preservare il dialetto

## 🧩 Contenuto del repository

Qui troverai:

*   File HTML/CSS/JS del sito
*   Script per collegarsi al database
*   Risorse varie (icone, immagini, ecc.)
*   Database (ancora da definire)
